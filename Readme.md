# My Project's

```yaml
Project
│
├── 2022
│   ├── Documentation
│   ├── Designs
│   ├── SourceCode
│   ├── Resources
│   └── Tests
│
├── 2023
│   ├── Documentation
│   ├── Designs
│   ├── SourceCode
│   ├── Resources
│   └── Tests
│
└── Archive
    ├── ProjectArchive_2020
    ├── ProjectArchive_2021
    └── ... 
```
